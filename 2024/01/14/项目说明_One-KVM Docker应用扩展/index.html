安装Docker

玩客云内部储存空间为8G，如果要使用Docker应用请注意储存空间使用情况。

本文所写Docker应用无需额外设置都可以与PiKVM共存，不必为此修改PiKVM配置文件。

安装docker-compose

添加执行权限：chmod +x docker_compose_manager.sh

./docker_compose_manager.sh install

./docker_compose_manager.sh uninstall

玩客云安装OpenWRT（Docker版）

系统环境：Armbian 22.11.0-trunk Jammy Linux onecloud 5.10.149-meson

Docker版本：Docker version 24.0.5, build 24.0.5-0ubuntu1~22.04.1

OpenWRT默认账号和密码：root/root
